Eve Vogelein Portfolio Website - Deployment Notes

This folder contains a static portfolio website.

Entry point:
- index.html

How to deploy:
- Upload the full contents of this folder to any static hosting provider.
- Keep all folders and files together exactly as they are.
- The website does not require a build step.
- The CV download button links to CV_Eve_2026_Stand_Juli.pdf in the same folder.

Important:
- The site uses external services for Tailwind, Google Fonts, external logo images, the Harvard badge image, and the embedded Alphabees chat.
- The CV contains Eve's email address and mobile phone number.

Suggested hosting options:
- Netlify
- Vercel
- IONOS / Strato / other classic webspace
- Any static web host that can serve an index.html file
